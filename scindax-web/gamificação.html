Módulo 01 / 03

1 Preparação do Ambiente de Borda

Recompensa +100 XP

Para começarmos com o pé direito, precisamos inicializar nosso SDK da Scindax e preparar a CDN de borda que trará os modelos leves de IA direto para o ecossistema do navegador. Copie o script de bootstrapping abaixo e adicione na tag <head> do seu arquivo HTML.

Terminal Bash
$ npm install @scindax/pdf-core @edge-ai/onnx-runtime

Ao baixar essas dependências, você garante que o fatiador de PDFs e os binários em WebAssembly (Wasm) estejam prontos para realizar inferências locais no hardware do cliente.

Módulo 03 / 03

3 Deploy de Produção Serverless

Recompensa +100 XP

O último passo é disparar nossa função estruturada para os servidores de borda globais usando nossa CLI de implantação. Este comando consolida os módulos do WASM e cria uma camada de controle de cache estático persistente.

Deployment Console
$ scx deploy --prod --edge-runtime="v8"

Isso fará o deploy global na nossa malha de borda CDN.
