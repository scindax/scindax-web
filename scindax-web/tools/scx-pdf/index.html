👋 Bem-vindo ao SCX PDF Tool

Esta é uma ferramenta experimental do Scindax Lab, em fase de testes. Todo o processamento dos seus PDFs acontece localmente, no seu navegador — nenhum arquivo é enviado para nossos servidores.

Há um limite de 20 processamentos por dia (com verificação de segurança via Cloudflare Turnstile) para manter a ferramenta gratuita para todos.

Podemos coletar dados anônimos de uso (como número de páginas processadas e o texto de amostra enviado à IA para sugestão de nome) para melhorar nossos produtos. Dados sensíveis (CPF, endereços, valores bancários) nunca são coletados ou armazenados.

Nenhum cadastro ou e-mail é necessário para usar esta ferramenta.
